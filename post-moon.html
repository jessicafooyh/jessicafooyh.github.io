Moon Landing: DQN

Learning through exploration and exploitation

Posted by Jessica

This was one of our final project ideas for the grad-level computer science class at the Toyota Technological Institute of Chicago on probabilistic artificial intelligence - to train a moon lander to learn to land on a landing pad, digitally (haha). This is an environment provided on the OpenAI Gym, which is a great resource for developing Reinforcement Learning algorithms (last I checked, they included a section on Robotics!). For this implementation, I used source code from keras-rl, which works with OpenAI Gym out of the box, so it was relatively easy to implement. In general, reinforcement learning allows a learning agent to interact with its environment to achieve a pre-specified goal (and reward), which requires the agent to both exploit what it already knows (experience) and explore uncharted territory to make better action selections in future (Sutton and Barto, 2012). For this model specifically, I used Deep Q Learning with epsilon-greedy action selection. For the network architecture, I used three hidden layers and ReLU activation functions, followed by a linear output layer. Since I don't have a GPU (yet), this model is not optimised by architecture or parameters etc. I used a learning rate of 0.002 and an epsilon decay rate of 0.975. This wasn't very difficult to implement, given the abundance of resources online for OpenAI Gym. However, my exposure to RL is still quite nascent, and I hope to explore it further with different environments!

The graph on the left shows the total reward for each episode, and the graph on the right shows the 10-episode moving average. As the agent learns to navigate the environment, the reward increases, exponentially at first. Over time, there's still some variation in the reward as the agent explores the environment, but the reward somewhat stabilises. The video below shows how the agent fails to meet the target in the first few episodes, resulting in negative rewards, but starts to learn with more training, and eventually lands successfully on the pad!
