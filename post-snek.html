Snek

A classic game in Java

Posted by Jessica

In true Wharton snek fashion, I did the bare minimum for my CIS 120 final project - something that wouldn't take too much time, but would still give me the score I needed for this class. So I did Snake, the classic game that we obsessed over on our Nokia phones, with help from the starter code provided. I don't remember much from my CIS classes, but they gave me a solid understanding of object-oriented programming and unit testing - both of which, I've come to realise, are quite important in productionising code even in analytics type of projects.

Since I'm such a snek, I've tweaked my class submission a little - do I finally qualify for a GS super day?
